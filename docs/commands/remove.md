# wt remove

Remove a git worktree and delete its working directory

## Syntax

```bash
wt remove <worktree> [options]
```

## Arguments

**`<worktree>`**  
Worktree identifier (branch name or path)

## Options

`-f, --force`  
Force removal even with uncommitted changes or locks

`-o, --output <Human|Json>`  
Output format (human or json) [default: Human]

`-v, --verbose`  
Show detailed diagnostic information
